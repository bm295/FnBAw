# User Stories

Status: Done

## Dashboard workflows

- As an owner, I want to see today’s revenue and order count so that I can understand current business performance.
- As a manager, I want to see low-stock alerts on the dashboard so that I can prevent service interruptions.
- As a shift lead, I want to see recent orders so that I can quickly investigate service issues.

## Menu workflows

- As a manager, I want to add a menu item with name, category, price, and availability so that staff can sell it correctly.
- As a manager, I want to mark an item unavailable so that cashiers do not sell items the kitchen cannot prepare.
- As kitchen staff, I want to view available menu items so that I know what can be prepared during service.

## Inventory workflows

- As inventory staff, I want to view items below reorder level so that I can prioritize purchasing.
- As inventory staff, I want to adjust stock after receiving ingredients so that quantities stay accurate.
- As a manager, I want to review inventory exceptions so that I can control waste and shortages.

## Order workflows

- As a cashier, I want to create an order quickly so that guests can be served without delay.
- As a manager, I want to review today’s orders so that I can monitor service volume.
- As kitchen staff, I want to see incoming order demand so that I can prepare items in the right sequence.

## Reporting workflows

- As an owner, I want to review sales by date range so that I can track performance trends.
- As a manager, I want to export operational reports so that I can share shift summaries with leadership.
- As inventory staff, I want inventory reports to show reorder needs so that purchasing is based on current data.
